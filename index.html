NotPacman

by: Annie Bian

This game was made for my level 2 project at The League of Amazing Programmers.
Click here to download and play!

1. Why I Made This Game

I decide to make this game because it seems fun to make and interesting. At first I was thinking about the game Color, but somebody else already made it. Then i thought about the dinosaur game on google chrome, but then I just don't feel like it. So in the end i made NotPacman.

2. What I Learned

One suggestion I got is to make the game harder over time, to make the game more interesting. So then I make the "ghosts", and the pac-man speed up every 10 seconds. After I learned that if the speed is too high, the ghosts start to teleport. When the ghosts teleport they can teleport out side of the boundary, which is making them disappearing. So Then i made a highest speed the ghost can go, so they wont be able to escape.

3. Architecture of My Game

I have a main class, object manager class, and panel (display) class, and lastly I have a obj class that four class extended from, which are the ghosts, pac-man, walls, and points. When ghost touch pac-man, the panel will switch from game state to end state. The wall stop the pac-man from moving. Finally, when the point and pac-man touch, the p[point teleport away and add one to the int Score.

4. Technical Implementation

one part of the technical implementation that I am proud of is how the pac-man move. In the beginning, i have 4 boolean for the four direction and one of them will be true when i press the corresponding key and the others will be false. First, I made two integer, newX and newY. Secondly i move the pac-man to the true direction using the newX and newY. Then I tested if the new position intersect with any walls. If it did then the real X, and Y don't move there, if its false then the pac-man will move there.

5. Most Challenging Part

Then most changing part in the game was to make the ghost move correctly. First I try to make the ghost go a random direction when they touched a wall, but it restricted the possibility of ghost to go in some area. Then I try to use intersections, but it don't work, it just make everything lag and glitchy. So in the end I made the ghost able to go though walls like how a ghost are supposed to do.

6. Future Fixes and Updates

I would make the game some what possible to beat and make the game automatically when all the point is collected.
